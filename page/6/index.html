redis 五种数据类型的基本使用

发表于 2019-12-08 更新于 2020-05-11 分类于 Redis Valine：
redis 中有5种基本的数据类型： * string * hash * list * set * zset 常用命令： * type：查看当前结构类型 * ttl：查看当前键的过期时间，-1代表永不过期 * flushall：清空所有库的数据 * save：强制保存 * del：删除指定的 key * rename：重命名 key 的名称 一、string string 数据类型是开发中使用最多的。 string 常用命令： * set：设置值 * setex：设置 key 的过期时间，单位为秒 * psetex：设置 key 的过期时间，单位为毫秒

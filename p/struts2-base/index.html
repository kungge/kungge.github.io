发表于 2020-02-01 更新于 2020-05-14 分类于 Java框架 阅读次数： Valine：

Struts2 是一个 MVC 框架。

Struts1 完全依赖于 Servlet 的 API 但是耦合性太高，被 Struts2 替代了。

Struts2 并不是在 Struts1 基础上设计的，两者没有直接的关系。

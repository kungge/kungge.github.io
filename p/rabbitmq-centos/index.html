P、常见问题

P.1 User can only log in via localhost

出现这个问题的原因是 RabbitMQ 从 3.3.0 版本开始禁止使用 guest/guest 权限通过除 localhost 以外的地址访问。

将 ebin/rabbit.app 中的内容 {loopback_users, [<<"guest">>]}, 改为 {loopback_users, []},，然后重启服务即可。
